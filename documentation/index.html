CSS Files and Structure

I have used set of CSS files in this template (under css folder)

  1. Colors - Contains the main 3 colors of the template which make it easy to change the full template color, by changing the hexadecimal color value (#b06ab3), and you can add your own color by copying one of the color files( For Example color-1.css) and change the color values in side the file and save it, Enjoy your own color now
  2. style.css - Contains all of the specific stylings for the template pages.
  3. start-page.css - Contains all of the specific stylings for intro page(index.html).
  4. bootstrap.css - Bootstrap css.
  5. font-awesome.min.css - Fontawesome Icons.
  6. owl.carousel.min.css - Contains all of the specific stylings for owl carousel silder
  7. flaticon.css - Contains all of the specific stylings for custom icons
  8. animate.css - Contains all of the specific stylings for animate library

style.css file contains all of the specific stylings for the page. The file is separated into sections using:

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

  9. js/jquery.easing.min.js - To Enable jQuery scrolling to specific section.
  10. js/jquery.min.js - To Enable jQuery.
  11. js/owl.carousel.min.js - To Enable owl carousel Plugin.
  12. js/wow.min.js - Animation Plugin.
  13. js/particles.js - Particles Plugin
  14. js/script.js - Template custom jQuery code
